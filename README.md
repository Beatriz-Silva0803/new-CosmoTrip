# 🚀 CosmoTrip - Space Exploration

[Português] | [English]

## 🇧🇷 Sobre o Projeto
O *CosmoTrip* é uma interface moderna de exploração espacial desenvolvida para apresentar destinos interplanetários de forma imersiva. Este projeto foca em uma experiência de usuário (UX) fluida e um design visual inspirado no cosmos.

### 🛠 Tecnologias Utilizadas
* *HTML5* - Estrutura semântica.
* *CSS3* - Estilização e layouts responsivos.
* *Git & GitHub* - Controle de versão.
